HTTP GET example

    
$http.get('//jsonplaceholder.typicode.com/posts').then((data) => {
    $scope.items = data.data;
});
USed http service to fetch online data, on success load the data to $scope data. Items data is data binded to table.

<tr ng-repeat="item in items">
<td> item.id  </td>
<td item.userId  </td>
<td> item.title </td>
</tr>
Items will hold http response data, which will be repeat and will display the data in table.
